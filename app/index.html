This prototype is to develop a library to achieve some of the concepts fo knockout (or maybge angular) to decouple the data model from the view model. The following functionality looks like required

  • Bind properties of an object to a node (span, div, other)
  • Handle some sort fo scope for nested objects
  • Enable looping through arrays
  • Allow event handlers for asynchronous updates
  • Listeners to update the view model when data changes
  • Looks like some really big library needed!
